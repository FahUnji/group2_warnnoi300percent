## Critical Anti-Patterns (do NOT repeat these)

- **Stage `.planning/*.md` files**: CRLF-only diff noise — always skip these in `git add`
- **Edit backend `.py` and expect live changes**: uvicorn --reload is blind to NTFS — always `docker restart` backend
- **Run `git add .` or `git add -A`**: too broad — will stage CRLF noise + unrelated files. Use explicit paths.
- **sessionStorage for cross-page flags**: document.referrer is simpler and more reliable for same-origin navigation state
