## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| Worktree files committed | `.claude/worktrees/` missing from `.gitignore`; fixed earlier | advisory | `workflow.use_worktrees=false` in config.json; never change |
| npm install EACCES on WSL NTFS | Fails on `/mnt/d/` paths | advisory | Use Docker or native Linux path |
| CRLF diff false positives | `.planning/*.md` show hundreds of modified lines — CRLF only | advisory | Never stage `.planning/` unless content was intentionally edited. Stage source files by name. |
| FastAPI error format | `HTTPException` with dict `detail` serializes as `{"detail": {...}}` | advisory | Frontend must use `err.detail?.message` not `err.message` |
| No get_current_user | This codebase has NO JWT per-endpoint auth — auth is ambient OAuth | advisory | Never add `Depends(get_current_user)` to any endpoint |
| Docker image stale after requirements.txt change | pip install layer cached; new packages missing at runtime | advisory | `docker-compose build --no-cache backend` after any requirements.txt edit |
| Excel sheet title invalid chars | openpyxl raises ValueError on `/ \ * ? : [ ]` in sheet name | advisory | Always sanitize: `re.sub(r'[\\/*?:\[\]]', '-', name)[:31]` |
