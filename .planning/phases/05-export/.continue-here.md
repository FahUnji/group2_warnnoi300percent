## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| uvicorn watcher blind to NTFS | `--reload` does NOT detect .py changes on WSL NTFS mount | advisory | Always `docker restart group2_warnnoi300percent-backend-1` after editing .py files |
| Docker CLI not in WSL | `docker` command not found in WSL shell | advisory | Use: `powershell.exe -Command "cd '...'; docker ..."` |
| CRLF diff false positives | `.planning/*.md` show hundreds of modified lines — CRLF only | advisory | Never stage `.planning/` unless content was intentionally edited |
| Edit bleed between functions | Stray line from one function leaked into next during surgical edit — caused runtime NameError | advisory | After any function-boundary edit, syntax-check immediately: `python3 -c "import ast; ast.parse(open('...').read())"` |
