## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| uvicorn watcher blind to NTFS | `--reload` does NOT detect .py changes on WSL NTFS mount | advisory | Always `docker restart group2_warnnoi300percent-backend-1` after editing .py files |
| Docker CLI not in WSL | `docker` command not found in WSL shell | advisory | Use: `powershell.exe -Command "cd '...'; docker ..."` |
| CRLF diff false positives | `.planning/*.md` show hundreds of modified lines ‚Äî CRLF only | advisory | Never stage `.planning/` unless content was intentionally edited |
| Excel sheet title invalid chars | openpyxl raises ValueError on `/ \ * ? : [ ]` in sheet name | advisory | Always sanitize: `re.sub(r'[\\/*?:\[\]]', '-', name)[:31]` |
| Flex truncation needs min-width:0 | text-overflow:ellipsis won't work on flex child without min-width:0 on container | advisory | Always add `min-width:0; flex:1` to text container inside flex row |
